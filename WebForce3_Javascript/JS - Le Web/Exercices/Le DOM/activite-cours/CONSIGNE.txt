Les sélecteurs js dom : 

1- Sélecteur par identifiant :

Pour sélectionner un élément avec un identifiant spécifique, utilisez la syntaxe suivante :

const element = document.getElementById('id');

2- Sélecteur par nom de balise :

Pour sélectionner tous les éléments d'une balise spécifique, utilisez la syntaxe suivante :

const elements = document.getElementsByTagName('tagname');

3- Sélecteur par nom de classe :

Pour sélectionner tous les éléments ayant une classe spécifique, utilisez la syntaxe suivante :

const elements = document.getElementsByClassName('classname');

4- Sélecteur par sélecteur CSS :

Pour sélectionner un ou plusieurs éléments à l'aide d'un sélecteur CSS, utilisez la fonction querySelector pour un seul élément ou querySelectorAll pour plusieurs éléments :

const element = document.querySelector('css-selector');
const elements = document.querySelectorAll('css-selector');

5- Sélecteur par attribut :

Pour sélectionner les éléments ayant un attribut spécifique, utilisez la syntaxe suivante :

const elements = document.querySelectorAll('[attribute="value"]');

6- Sélecteur d'enfant :

Pour sélectionner un élément enfant spécifique d'un élément parent, utilisez la propriété children ou la méthode querySelector sur l'élément parent :

const parentElement = document.querySelector('parent-selector');
const childElement = parentElement.querySelector('child-selector');
